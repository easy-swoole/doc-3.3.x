## 准备词库


服务启动的时候会一行一行将数据读出来，每一行的第一列为敏感词，其它列为附属信息。

`官方不提供词库，只提供工具`

文件内容示例：

```
php,是世界上,最好的语言
java
golang
程序员
代码
逻辑
```

## config

#### wordBanks

词库路径配置

#### processNum 

words-match进程数，默认3

#### maxMem 

每个进程最大占用内存，默认512M

#### separator

每行词信息的分隔符，默认逗号
